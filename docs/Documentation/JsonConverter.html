JsonConverter [Class]

Description

The JsonConverter class is responsible for converting XML structures into JSON format.

It takes a pointer to the root of a parsed XML tree (represented by Node<std::string>*) and traverses it to produce a valid JSON string representation.

Static Methods

static std::string convert(Node<std::string>* root)
Converts the given XML tree node structure into a JSON string.
Parameters
  • root: A pointer to the root node of the XML tree.
Returns
A std::string containing the resulting JSON.
